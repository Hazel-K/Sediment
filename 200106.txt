* Version Control
repository = 코드 저장 폴더
git = 코드의 변경사항을 추적하는 시스템
commit = 변경 사항 기록 (프로젝트의 상황 창)
branch = 나뭇가지. 원본인 마스터 브랜치를 복사하여 새로운 기능을 시험할 때 쓰는 복사본
