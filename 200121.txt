* 변수 설명
constant (상수)
형태 : const 상수 이름 : 상수 상세;
주로 사용할 변수 지정. 상수를 지정한 후, 이 상수에 대입을 할 수 없다[값 변경 불가]

let
형태 : let 변수 이름 : 변수 상세;
가변적인 변수를 지정한다.

var
형태 : var 변수 이름 : 변수 상세;
3년 전에는 const, let이 없었음 그 때 주로 사용 let과 비슷함


* 코멘트(주석 설명)
형태 : // 주석 내용
'여기 쓴 건 보지도 건들지도 말라'라고 컴퓨터에게 이야기하는 셈

* string 타입
형태 : "텍스트 내용"
해당 내용을 텍스트로서 인식시킴

* Nunber 와 float
정수와 소수

* Array
형태 : const 변수 이름 = ["값1", "값2", "값3" ... ];
console.log(변수 이름[#]) <이러면 #에 들어간 숫자 위치에 해당하는 값을 보여줌, 0부터 시작>
데이터 저장 방식, list 형식으로 제작

* object
형태 : const 변수 이름 = {속성1:"값1", 속성2:"값2", 속성3:"값3" ... };
colsole.log(변수 이름.속성1) <이러면 변수의 속성 1에 해당하는 값을 보여줌>
데이터 저장 방식

*브라켓 [ ]과 컬리 브라켓 { }

* Camel Case
변수 이름 지정 방식, 변수 이름의 시작은 반드시 소문자로, 그 이후에 띄어 써야 할 경우 띄어쓰기 대신 대문자로 적는다. JS에서는 변수명을 띄어 쓸 수 없기 때문

* function(함수)
형태 : function 함수 이름( ) { }

* 함수 속 argument 넣기
형태 1 : function 함수 이름(속성1, 속성2) {console.log("hello", 속성1, "you are", 속성2)}
형태 2 : function 함수 이름(속성1, 속성2) {console.log(`hello ${속성1} you are ${속성2}`);}
