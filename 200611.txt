Eclips 초기 설정
Window.Preferences - Filter에 'enc'입력
Content Types - Default encoding을 UTF-8입력 후 업데이트
workspace - encoding을 UTF-8 로 설정 (workspace 폴더 변경할 때마다 설정)

Java 패키지 이름 설정
전 세계에 유일해야 함.
구분하기 위해 길게 설정 ex) com.naver.projectname.object.java
패키지명에 대문자가 들어가서는 안된다.

Java 클래스 이름 설정
class는 객체(instance)를 만드는 틀을 의미
기능명을 이해하기 쉽게 설정
띄어쓰기를 대문자로, 첫 시작은 무조건 대문자로

Java 메인 메소드(함수) 설정
public static void main(String[] args){ } 메인 메소드 기본 형태. 항상 같은 형태로 제작

Java 언어 이해
첫글자 대문자면 class
소괄호 없으면 변수
소괄호 있으면 method
